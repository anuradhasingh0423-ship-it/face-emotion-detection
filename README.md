# 😄 Face Emotion Detection

Real-time Face Emotion Detection using Deep Learning, OpenCV, and CNN.

## 🚀 Features
- Detects human emotions from webcam
- Real-time face detection
- CNN-based emotion classification

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV

## 📂 Dataset
FER-2013 dataset

## ▶️ How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Train model
python train.py

### 3. Run app
python app.py

## 🎯 Output
Detects:
- Happy 😊
- Sad 😢
- Angry 😠
- Surprise 😲
- Neutral 😐
